我的第一篇博客：GitHub Pages 搭建记录

2026-06-07 / 博客搭建 / GitHub Pages

今天我成功用 GitHub Pages 搭建了自己的第一个个人博客。 这个博客目前还很简单，但它已经可以作为一个正式的学习记录空间使用了。

我先创建了 GitHub 仓库，然后新建了首页文件 index.html， 接着通过 GitHub Pages 把它部署成一个可以访问的网站。

后来我把首页设计成了复古档案馆风格，左边放个人信息，中间放文章列表， 右边放标签、归档、更新记录等工具区。

目前的博客结构

flyhwjf/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── post/
│   └── my-first-blog.html
└── assets/
    └── images/

这样拆分以后，首页负责内容结构，CSS 文件负责样式，JavaScript 文件负责搜索等交互功能。 后面如果文章变多，也不会全部挤在一个文件里。

后续计划

接下来我准备继续添加 Python 学习笔记、英语六级作文积累、专业课程笔记等内容。 这个博客会慢慢变成我的学习档案馆。
